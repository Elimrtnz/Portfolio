My Story

Hello and welcome to my portfolio! My name is Elias Martinez and I am a web developer with a focus on database management and server side development.

My coding journey began in college when I took a programming introduction course that utilized Windows Visual Basic. The final exam for the class required students to make a coin flip game where the user had to guess whether the coin would land on heads or tails. Even though the game was simple, I was still very proud of the program I created and signed up for more programming classes.

Ultimately, taking night classes while working full time was not helping me reach my goals fast enough, so I applied to LearningFuze an accelerated web development course. During my time there I learned multiple coding languages, built many web applications and was exposed to a real workplace coding environment.

Currently, I am looking for a role that will allow me to further utilize and advance my knowledge of web development. I work primarily with PHP as my language of choice, but I am always looking to expand my skill set. Please feel free to contact me for any projects you may be interested in collaborating on.

Student Grade Table

A simple content management system for record keeping.

The Student Grade Table is an application made with Bootstrap, Javascript, PHP, and MySQL. Users are able to create, read, update, and delete data from the database. Additional features include calculating average grade, form validation, and sorting the list based on name, course, or grade.

MK VS DC Memory Match

A two player memory match game with a MK VS DC theme!

MK VS DC Match is a memory match game featuring characters from Motral Kombat and DC Universe. I used Javascript/jQuery, HTML, and CSS to create the game functionality and animations. Features include randomized cards, attack/damage system, sound effects, and win/lose conditions.
